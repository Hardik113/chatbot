About us

Chatbot is a computer program which conducts a conversation via auditory or textual methods. Such programs are often designed to convincingly simulate how a human would behave as a conversational partner, thereby passing the Turing test. Chatterbots are typically used in dialog systems for various practical purposes including customer service or information acquisition.

The College bot project is built using artificial intelligence algorithms that analyses user’s queries and understand user’s message. This System is a web application which provides answer to the query of the student. Students can chat using any format there is no specific format the user has to follow.
